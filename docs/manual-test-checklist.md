# hueman 수동 테스트 체크리스트

자동화로 못 거르는 부분(웹캠 감정 인식, 포토 모자이크, 분기 흐름)을 사람이 직접 확인하는 QA 시나리오. 친구에게 테스트를 맡길 때 그대로 넘겨도 됩니다.

## 사전 준비
- [ ] `npm install` → `npm run dev` → 브라우저에서 표시된 `localhost` 주소 열기
- [ ] 카메라 있는 기기 + 적당한 밝기의 조명
- [ ] `D` 키로 우상단 **테스트 패널**을 켜 두면 내부 상태를 보며 확인 가능
- [ ] 개발자도구 콘솔을 열어 에러가 없는지 함께 확인

---

## 1. 인트로 / 진입
- [ ] 인트로에 제목 **hueman**, 설명, 버튼 2개(**카메라 켜고 시작** / **카메라 없이 시작**), **작가 노트** 링크 표시
- [ ] 작가 노트 클릭 → 오버레이에 4개 섹션(무엇을 / 어떻게 플레이 / 뒤에 있는 생각 / AI 사용)
- [ ] 오버레이 닫기: ✕ 버튼, 바깥 영역 클릭, `Esc` 모두 동작
- [ ] **카메라 없이 시작** → 바로 오프닝 진입
- [ ] **카메라 켜고 시작** → 권한 프롬프트 → 허용 시 우하단 카메라 미리보기(거울) 표시
- [ ] `Space` 또는 `Enter` → "카메라 켜고 시작"과 동일하게 시작

## 2. System A — 스토리 / 분기
- [ ] 오프닝(노란빛) → 클릭 · `Space` · `Enter` 로 진행
- [ ] 선택 장면: 버튼 2개가 뜨고, **클릭**과 **`1`/`2`(또는 `←`/`→`)** 둘 다 선택됨
- [ ] 유아기 분기: 운다 / 참는다
- [ ] 유년기 분기: 손을 내민다(따뜻함) / 혼자 일어선다
- [ ] 청소년기 고백: 사귄다 / 거절 — 패널 flags에 `dating` 확인
- [ ] 청소년기 담배: **거절(모범)** → 공부·수능·대학 경로
- [ ] **배드엔딩 경로**: 담배 **받아들인다** → **집을 나간다** → 가족상담 → 곧바로 `bad_ending` → 작품 엔딩 (서사는 짧아도 마무리 동일)
- [ ] 청년기 의견 충돌: 내 길(conflict) / 부모님 길(adapt) — 패널 flags 확인
- [ ] 중장년 결혼 / 비혼 → 노년기
- [ ] 노년기: `warmth ≥ 2` 면 따뜻한 임종, 아니면 외로운 임종 (패널 flags로 분기 예측)
- [ ] 작품 엔딩 내레이션 "…그 모든 감정이, 너였다" → 클릭 시 결과 화면
- [ ] **5회 이상 서로 다른 분기로 플레이 — 모든 경로가 결국 작품 엔딩으로 닫힘**

## 3. System B — 실시간 감정 → 색
- [ ] 카메라 ON 상태에서 **표정을 바꾸면 ~1초 후 화면 색이 바뀜**
  - [ ] 웃음 → 노랑(기쁨) / 찡그림 → 빨강(분노) / 놀람 → 초록 / 무표정 → 검정 / 겁먹은 표정 → 주황(불안)
- [ ] 테스트 패널 **카메라 감정**: `감지 X → 적용 Y` 가 실시간 갱신
- [ ] 얼굴을 가리면 → `얼굴 없음` 표시 + 화면이 **장면 색(fallback)** 으로
- [ ] 우하단 미리보기가 거울처럼 좌우 반전
- [ ] **카메라 OFF**로 시작하면 패널에 `OFF · 장면 색`, 장면별 색만 적용

## 4. System C — 포토 모자이크
- [ ] 카메라 ON으로 플레이하는 동안 패널 **모자이크: 타깃 ✓ · 타일 N장** 의 N이 증가
- [ ] 결과 화면에 **모자이크 캔버스** 표시 — 멀리서 보면 얼굴, 가까이서 보면 표정 타일(감정색 틴트)
- [ ] **이미지 저장** 버튼 → `hueman-<감정>.png` 다운로드
- [ ] **카메라 OFF**로 끝까지 → 모자이크 없이 placeholder 안내문 표시
- [ ] 아주 빠르게 클릭해 타일이 거의 안 쌓인 경우에도 깨지지 않고 placeholder

## 5. 결과 / 감정 집계
- [ ] 카메라 ON: **실제 표정** 기반 최빈 감정이 표시됨
- [ ] 카메라 OFF: **장면 감정** 기반 집계
- [ ] 위로 메시지가 표시된 최빈 감정과 의미상 일치 (1·2위가 비슷하면 "복합 감정" 메시지)

## 6. 테스트 패널
- [ ] `D` 키로 끄기/켜기
- [ ] 현재 씬 id, 단계/타입, 카메라 감정, 장면 감정, 누적 flags, 실시간 집계, 경로 트레일 표시

## 7. 엣지 케이스 / 견고성
- [ ] 카메라 **권한 거부** 후에도 체험이 끝까지 진행됨
- [ ] 카메라 **없는 기기**에서 "카메라 켜고 시작" → 멈추지 않고 장면 색으로 fallback
- [ ] 어두운 조명 → 감지가 무감각(neutral)에 머물 수 있음 (패널 `감지` 값으로 확인)
- [ ] 빠른 연타로 진행해도 흐름이 깨지지 않음
- [ ] 콘솔에 에러 없음

## 8. 회귀 테스트 (자동)
- [ ] `npm test` → **46 passing (10 files)**
- [ ] `npm run build` → 성공
